## 新增需求

### 需求:登录失效时提供跳转按钮

当系统检测到登录失效时，必须提供"打开网站登录"按钮，让用户一键跳转到 h-comic.com。

#### 场景:设置面板显示跳转按钮

- **当** 登录状态为"未配置登录信息"或"登录信息已过期"
- **那么** 设置面板登录状态旁必须显示"去登录"按钮

#### 场景:收藏夹登录失效弹窗

- **当** 用户访问收藏夹且登录失效
- **那么** 系统必须显示自定义对话框，包含"打开网站登录"按钮和操作指引

#### 场景:点击按钮打开浏览器

- **当** 用户点击"打开网站登录"或"去登录"按钮
- **那么** 系统必须使用 `webbrowser.open("https://h-comic.com")` 打开浏览器

### 需求:操作指引

当显示登录失效提示时，必须提供清晰的操作步骤指引。

#### 场景:收藏夹弹窗显示操作指引

- **当** 收藏夹登录失效弹窗显示
- **那么** 弹窗必须包含以下步骤指引：
  1. 点击按钮打开网站登录
  2. 登录后复制 curl 命令到设置面板
  3. 点击"应用登录信息"按钮
