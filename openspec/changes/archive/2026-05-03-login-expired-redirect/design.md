## 上下文

HComic Downloader 是一个 tkinter GUI 应用，用于从 h-comic.com 下载漫画。用户需要通过 Cookie 和 User-Agent 进行身份验证。当前登录失效时，程序只显示警告消息，用户需要手动完成整个重新登录流程。

## 目标 / 非目标

**目标：**
- 在登录失效时提供一键跳转到网站的按钮
- 在多处（设置面板、收藏夹弹窗）提供此功能
- 提供清晰的操作指引

**非目标：**
- 自动登录或自动获取 Cookie
- 修改登录验证机制
- 支持其他网站的登录跳转

## 决策

### 1. 使用 `webbrowser.open()` 打开浏览器

**选择**: 使用 Python 标准库 `webbrowser.open("https://h-comic.com")`

**替代方案**: 
- 使用 `subprocess` 调用系统命令（如 `start`、`open`、`xdg-open`）
- 在程序内嵌浏览器

**理由**: `webbrowser` 是 Python 标准库，跨平台兼容，无需额外依赖，代码简洁。

### 2. 使用自定义 Toplevel 对话框替代 messagebox

**选择**: 创建 `LoginExpiredDialog(tk.Toplevel)` 类

**替代方案**: 
- 使用 `messagebox.showwarning()`（不支持自定义按钮）

**理由**: tkinter 的 messagebox 不支持添加自定义按钮，必须使用 Toplevel 创建自定义对话框。

### 3. 按钮位置：组合方案

**选择**: 在多处添加按钮
- 设置面板登录状态旁（始终可见）
- 收藏夹登录失效弹窗中（上下文相关）

**替代方案**: 
- 只在设置面板添加
- 只在弹窗中添加

**理由**: 组合方案确保用户在不同场景下都能看到登录入口，提高可发现性。

## 风险 / 权衡

| 风险                      | 缓解措施                                       |
| ------------------------- | ---------------------------------------------- |
| 用户点击按钮后忘记回来粘贴 curl | 在对话框中提供清晰的步骤指引                 |
| 不同系统的浏览器行为差异  | `webbrowser.open()` 已处理跨平台差异           |
| h-comic.com 登录页面 URL 变化 | URL 硬编码为首页，登录入口在页面上显眼位置 |
