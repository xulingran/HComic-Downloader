## 新增需求

### 需求:moeimg 必须提供仅显示中文筛选

moeimg 来源必须显示默认关闭的“仅显示中文”开关。筛选状态必须作为搜索上下文的独立维度参与新查询、翻页、缓存和相邻页预加载；切换筛选必须从第一页重新查询。

#### 场景:moeimg 显示中文开关
- **当** 用户切换到 moeimg 来源
- **那么** 搜索界面必须显示“仅显示中文”开关
- **且** 未主动开启时禁止携带语言筛选参数

#### 场景:筛选上下文隔离
- **当** 同一个 moeimg 查询分别以开启和关闭中文筛选执行
- **那么** 两者必须使用不同缓存上下文
- **且** 翻页和相邻页预加载必须继续携带各自的筛选状态

### 需求:默认浏览必须优先使用服务端中文分页并安全回退

moeimg 空关键词默认浏览开启中文筛选时，系统必须优先通过已注入系统代理的 parser Session 请求 `/spa/language/chinese`，并直接采用服务端分页。只有该请求失败或响应结构不可解析时，系统才必须回退 `/spa/latest-manga`；合法空结果禁止触发回退。

#### 场景:中文端点成功
- **当** moeimg 空关键词第 N 页开启 `language_filter='chinese'`
- **那么** parser 必须请求 `/spa/language/chinese?page=N`
- **且** 禁止同时请求 `/spa/latest-manga`
- **且** 必须采用中文端点返回的分页信息

#### 场景:中文端点失败后回退
- **当** `/spa/language/chinese` 发生网络、HTTP、JSON 或响应结构错误
- **那么** parser 必须使用同一 Session 请求 `/spa/latest-manga?page=N`
- **且** 最近更新回退也失败时必须按现有错误路径向上报告

#### 场景:合法空结果不回退
- **当** 中文端点返回结构合法但 `manga_list` 为空
- **那么** 系统必须返回空中文结果及其分页
- **且** 禁止回退到包含非中文漫画的最近更新接口

### 需求:moeimg 列表必须透传语言元数据

moeimg parser 必须从搜索列表项提取规范化的 `language` 并写入 `ComicInfo.language`。缺失或空语言必须保持为未确定值，禁止默认标记为中文。

#### 场景:列表项包含中文语言
- **当** moeimg 列表项包含 `language: "chinese"`
- **那么** 返回前端的 `ComicInfo.language` 必须为 `chinese`

#### 场景:列表项缺失语言
- **当** moeimg 列表项不含 language 或值为空白
- **那么** `ComicInfo.language` 必须保持为空
- **且** 禁止仅因筛选已开启而补写 `chinese`

### 需求:非默认搜索必须执行占位式当前页语言过滤

moeimg 关键词、作者和标签搜索开启中文筛选时必须保持现有服务端请求与分页，并对当前返回页执行语言判断。已知语言且规范化后不为 `chinese` 的条目必须显示为屏蔽占位符；缺失语言的条目必须保持可见。

#### 场景:关键词结果包含非中文漫画
- **当** moeimg 中文筛选开启且关键词搜索结果的 language 为 `japanese`
- **那么** 该漫画必须以屏蔽占位符呈现
- **且** 系统禁止修改服务端返回的总页数或总数

#### 场景:语言比较忽略大小写与空白
- **当** 返回语言为 ` Chinese ` 或其他仅大小写、首尾空白不同的中文值
- **那么** 该漫画必须作为中文正常显示

#### 场景:缺失语言保持可见
- **当** 中文筛选开启但 moeimg 返回项缺失 language
- **那么** 该漫画必须保持正常卡片呈现
- **且** 禁止为判断语言而逐项请求详情

#### 场景:语言筛选独立于标签过滤总开关
- **当** “仅显示中文”开启而全局标签过滤关闭
- **那么** 已知非中文的 moeimg 漫画仍必须呈现为屏蔽占位符
