## ADDED Requirements

### 需求:关键词搜索
系统必须支持按关键词、作者、Tag 三种模式搜索漫画，搜索在后台线程执行，结果通过主线程回调展示。

#### 场景:执行搜索
- **当** 用户输入关键词并点击搜索或按回车
- **那么** 禁用搜索按钮，在后台线程调用 parser.search()，完成后在主线程渲染结果

#### 场景:搜索失败
- **当** parser.search() 抛出异常
- **那么** 恢复搜索按钮状态，显示错误消息对话框

### 需求:翻页导航
系统必须支持上一页、下一页和跳转到指定页码，翻页操作在后台线程执行。

#### 场景:上一页/下一页
- **当** 用户点击翻页按钮且当前不在边界页
- **那么** 更新页码，在后台线程加载对应页的结果

#### 场景:跳转指定页码
- **当** 用户点击页码标签
- **那么** 弹出输入对话框，用户输入有效页码后跳转

#### 场景:翻页按钮状态
- **当** 当前在第 1 页或最后一页
- **那么** 对应的翻页按钮必须禁用

### 需求:来源切换
系统必须支持在 h-comic 和 moeimg 之间切换数据来源，切换时清空当前结果和状态。

#### 场景:切换来源
- **当** 用户选择新来源且没有下载任务进行中
- **那么** 更新 parser 来源，清空搜索结果和选择状态，重置分页

#### 场景:下载中禁止切换
- **当** 下载任务进行中用户尝试切换来源
- **那么** 阻止切换并显示提示

### 需求:收藏夹浏览
系统必须支持查看用户收藏夹（仅限支持收藏夹的来源），收藏夹加载在后台线程执行。

#### 场景:查看收藏夹
- **当** 用户点击收藏夹按钮且当前来源支持收藏夹
- **那么** 重置为第 1 页，在后台线程加载收藏夹

#### 场景:登录失效提示
- **当** 收藏夹接口返回需要登录
- **那么** 恢复之前的视图模式和页码，弹出登录过期对话框

### 需求:结果网格布局
系统必须根据窗口宽度动态计算卡片列数，窗口大小变化时重新排列卡片。

#### 场景:窗口缩放时重新布局
- **当** 窗口宽度变化导致可容纳列数改变
- **那么** 销毁旧卡片，按新列数重新渲染所有结果卡片

### 需求:moeimg 详情预取
系统必须在搜索结果展示后，后台预取 moeimg 来源漫画的详情（页数、图片列表等）。

#### 场景:结果包含 moeimg 漫画
- **当** 搜索结果展示后检测到 moeimg 漫画
- **那么** 在后台线程依次预取详情，每完成一个更新对应卡片的作者和页数显示

#### 场景:结果切换时取消预取
- **当** 新的搜索结果加载
- **那么** 增加预取 generation counter，后台线程检测到不匹配时终止
