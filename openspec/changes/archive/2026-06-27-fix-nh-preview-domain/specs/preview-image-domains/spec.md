## 修改需求

### 需求:预览图片域名白名单必须包含所有已启用来源的图片服务器

系统在下载预览图片前必须校验目标域名是否在白名单中。白名单必须覆盖所有已启用来源的图片 CDN 域名，包括 nh 来源的 `i.nhentai.net`（图片）和 `t.nhentai.net`（缩略图）。

#### 场景:nh 来源预览图片正常加载

- **当** 用户预览 nh 来源的漫画，图片 URL 域名为 `i.nhentai.net`
- **那么** 系统必须允许该 URL 通过域名白名单校验并正常下载图片

#### 场景:nh 来源缩略图正常加载

- **当** 系统请求 nh 来源的缩略图，URL 域名为 `t.nhentai.net`
- **那么** 系统必须允许该 URL 通过域名白名单校验并正常下载

#### 场景:未授权域名仍被拒绝

- **当** 预览图片 URL 的域名不在白名单中（如 `evil.com`）
- **那么** 系统必须拒绝该请求并抛出 `Domain not allowed` 错误
