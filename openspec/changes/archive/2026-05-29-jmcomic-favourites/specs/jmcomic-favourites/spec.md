## ADDED Requirements

### 需求:用户可以查看 jmcomic 收藏夹

系统必须允许已登录用户查看 jmcomic 收藏夹中的漫画列表，支持分页浏览。

#### 场景:成功获取收藏夹第一页

- **当** 用户已登录 jmcomic 且请求收藏夹第一页
- **那么** 系统返回收藏夹漫画列表和分页信息

#### 场景:收藏夹为空

- **当** 用户已登录 jmcomic 但收藏夹为空
- **那么** 系统返回空列表，needsLogin 为 false

#### 场景:未登录状态

- **当** 用户未登录 jmcomic 或登录已过期
- **那么** 系统返回 needsLogin 为 true

#### 场景:分页浏览

- **当** 用户请求收藏夹第 N 页
- **那么** 系统返回对应页的漫画列表和正确的分页信息

### 需求:前端支持多源收藏夹切换

系统必须在收藏夹页面提供来源选择器，允许用户在 hcomic 和 jmcomic 之间切换。

#### 场景:默认显示 hcomic 收藏夹

- **当** 用户进入收藏夹页面
- **那么** 系统默认显示 hcomic 收藏夹

#### 场景:切换到 jmcomic 收藏夹

- **当** 用户在来源选择器中选择 jmcomic
- **那么** 系统加载并显示 jmcomic 收藏夹

#### 场景:切换回 hcomic 收藏夹

- **当** 用户在来源选择器中选择 hcomic
- **那么** 系统加载并显示 hcomic 收藏夹
