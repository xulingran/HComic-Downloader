## ADDED Requirements

### 需求:用户可以检查 jmcomic 漫画的收藏状态

系统必须允许已登录用户检查指定漫画是否已收藏。

#### 场景:漫画已收藏

- **当** 用户检查已收藏漫画的收藏状态
- **那么** 系统返回 isFavourited 为 true

#### 场景:漫画未收藏

- **当** 用户检查未收藏漫画的收藏状态
- **那么** 系统返回 isFavourited 为 false

#### 场景:未登录状态

- **当** 用户未登录 jmcomic 或登录已过期
- **那么** 系统抛出认证错误

### 需求:用户可以将 jmcomic 漫画添加到收藏夹

系统必须允许已登录用户将指定漫画添加到 jmcomic 收藏夹。

#### 场景:成功添加收藏

- **当** 用户将未收藏漫画添加到收藏夹
- **那么** 系统返回 success 为 true

#### 场景:漫画已收藏

- **当** 用户将已收藏漫画再次添加到收藏夹
- **那么** 系统返回 success 为 true（幂等操作）

#### 场景:未登录状态

- **当** 用户未登录 jmcomic 或登录已过期
- **那么** 系统抛出认证错误

### 需求:用户可以将 jmcomic 漫画从收藏夹移除

系统必须允许已登录用户将指定漫画从 jmcomic 收藏夹移除。

#### 场景:成功移除收藏

- **当** 用户将已收藏漫画从收藏夹移除
- **那么** 系统返回 success 为 true

#### 场景:漫画未收藏

- **当** 用户将未收藏漫画从收藏夹移除
- **那么** 系统返回 success 为 true（幂等操作）

#### 场景:未登录状态

- **当** 用户未登录 jmcomic 或登录已过期
- **那么** 系统抛出认证错误
