# preview-loading-placeholder 规范

## 目的

定义漫画预览（阅读器）界面中**未加载页面的统一占位视觉**。阅读器整体强制深色背景（`#1a1a2e`），但翻页模式原先使用走主题变量的浅色 `Skeleton`，导致浅色主题下阅读器内出现刺眼的白色色块；滚动模式与翻页模式的占位实现也不一致。本规范将"加载中"占位统一为阅读器背景色 + 中心 spinner，明确传达加载语义；与"未进入视口（懒加载）"的二态区分避免满屏 spinner 喧闹。

本规范不涉及失败页占位（由 `preview-error-recovery` 规范管理）、整章加载态（`ReaderLoadingState`）、列表查询加载态（由 `list-loading-feedback` 规范管理）或启动屏 Skeleton 体系（由 `startup-skeleton-screen` 规范管理）。

## 新增需求

### 需求:加载中占位必须使用阅读器背景色与中心 spinner

当阅读器内单页处于"已发起图片请求、尚未完成"（`urlHash` 未就绪、非 `error` 态）时，占位元素**必须**以阅读器背景色（`#1a1a2e`）填充，并**必须**在视觉中心显示一个加载 spinner（旋转动画）。占位**禁止**使用浅色主题变量（`--bg-secondary`/`--bg-tertiary`）作为底色，**禁止**在阅读器内显示近白色或主题相关的色块。该规则对滚动模式与翻页模式同时生效，两模式的"加载中"占位**必须**视觉一致。

#### 场景:翻页模式加载中占位使用阅读器背景色

- **当** 阅读器处于翻页模式（单页/双页）且当前页图片请求已发起但未完成
- **那么** 占位元素的背景色为 `#1a1a2e`，元素中心显示一个 spinner，**禁止**出现浅色主题变量驱动的色块

#### 场景:滚动模式加载中占位与翻页模式视觉一致

- **当** 阅读器处于滚动模式且某页已进入视口（或被标记为 `priority`）但图片请求未完成
- **那么** 该页占位的背景色与 spinner 样式**必须**与翻页模式的"加载中"占位完全一致

### 需求:加载中占位必须保持漫画页比例

为避免图片加载完成时占位高度突变导致阅读区跳动，加载中占位**必须**维持 `aspect-ratio: 3/4`（与现有占位比例一致），**必须**填满父容器允许的宽度，且**禁止**因 spinner 而改变占位元素的总尺寸。

#### 场景:加载中占位保持 3/4 比例

- **当** 任一模式的"加载中"占位被渲染
- **那么** 占位元素的 `aspect-ratio` 为 `3/4`，且 spinner 居中显示于该比例的元素内部，不影响元素外层尺寸

### 需求:滚动模式未进入视口的页面必须保留横纹懒加载占位

滚动模式下，未进入视口（`IntersectionObserver` 未触发 `isVisible`）且非 `priority` 的页面**必须**继续显示现有的 `repeating-linear-gradient` 横纹占位（懒加载占位），**禁止**对这些页面显示 spinner。该横纹占位与"加载中"占位构成二态区分，避免满屏未加载页同时显示 spinner。

#### 场景:未进入视口的页面显示横纹而非 spinner

- **当** 滚动模式下某页尚未被 `IntersectionObserver` 标记为可见、且未被标记为 `priority`
- **那么** 该页显示 `repeating-linear-gradient` 横纹占位，**不**显示 spinner，**不**发起图片请求

#### 场景:进入视口后从横纹切换为加载中占位

- **当** 滚动模式下某页首次被 `IntersectionObserver` 标记为可见（或被标记为 `priority`）且图片尚未加载完成
- **那么** 该页占位从横纹切换为"加载中"占位（阅读器背景色 + spinner）

### 需求:加载中占位必须在 reduced-motion 下静态显示

遵循项目 `ui-animation` 规范的双层 reduced-motion 降级策略，加载中 spinner 在用户启用 `prefers-reduced-motion` 时**必须**退化为静态状态（停止旋转），但占位的背景色与 `aspect-ratio` **必须**保持不变，仍能传达"占位"语义。

#### 场景:reduced-motion 下 spinner 停止旋转

- **当** 用户启用 `prefers-reduced-motion` 且"加载中"占位被渲染
- **那么** spinner 停止旋转动画，但占位背景色（`#1a1a2e`）与 `aspect-ratio: 3/4` 保持不变
