# history-card-source-divider 规范

## 目的
定义历史记录页（`HistoryPage`）封面网格布局下，漫画卡片"来源/元信息行"与卡片主体（标题/作者）之间视觉分隔的能力规范。要求来源行作为视觉独立的 footer，与卡片主体之间保持正向垂直留白且必须有顶部分割线；禁止使用负边距把来源行拉向主体，禁止依赖背景色区分。仅约束封面布局，`detailed` 列表布局不受影响。

## 需求

### 需求:封面布局历史卡片的来源/元信息行必须作为视觉独立的 footer，与卡片主体正向分隔

历史记录页（`HistoryPage`）**封面网格布局**（`cardStyle === 'cover'`）下，每张漫画卡片由共享 `ComicCard`（渲染封面图 + 标题 + 作者的主体，主体用 `p-2` 内边距收尾）与一个紧随其后的"来源/元信息行"组成。该来源/元信息行**必须**作为视觉独立的 footer 区域，与卡片主体（标题/作者块）之间保持正向垂直留白，**且必须**有顶部分割线。

具体地：
- 来源/元信息行容器**禁止**使用负上边距（如 `-mt-1`）将其拉向卡片主体——负边距会吃掉 `ComicCard` 主体的下内边距，使来源行贴死作者行底部。
- 来源/元信息行容器**必须**使用正向上边距 + 正向上内边距留出与卡片主体的间距。
- 来源/元信息行容器**必须**渲染顶部分割线（`border-t`），分割线颜色**必须**复用项目边框分隔 token（`var(--border)`），以与 detailed 布局的行分隔（`border-b border-[var(--border)]`）视觉一致。
- 分割线**禁止**依赖背景色区分——因 `CoverCard` 渲染圆角容器而 footer 是其同级元素，背景色会在圆角下方形成视觉割裂的直角条带；只能用顶部分割线 + 留白表达独立性。

此需求仅约束封面布局。`detailed` 列表布局的来源标签是同一 `text-xs` div 内的内联 `<span>`（用 `·` 分隔），行间靠父容器 `border-b` 分隔，不受此需求约束。

#### 场景:来源行与卡片主体之间有正向垂直间距

- **当** `HistoryPage` 以 `cardStyle === 'cover'` 渲染历史卡片列表
- **那么** 每张卡片的来源/元信息行容器的上边距（`margin-top`）**必须**为正值（≥ 0），**禁止**为负值
- **且** 来源/元信息行容器的上内边距（`padding-top`）**必须**为正值（≥ 0），使来源文字与卡片主体底部之间有可见留白

#### 场景:来源行顶部必须有分割线

- **当** `HistoryPage` 以 `cardStyle === 'cover'` 渲染历史卡片列表
- **那么** 每张卡片的来源/元信息行容器**必须**渲染顶部分割线（`border-top` 宽度 ≥ 1px）
- **且** 分割线颜色**必须**为 `var(--border)`（项目边框分隔 token），**禁止**使用硬编码颜色
- **且** 分割线在深色模式与浅色模式下**必须**自动适配（因复用 `--border` CSS 变量）

#### 场景:来源行禁止依赖背景色区分独立区域

- **当** `HistoryPage` 以 `cardStyle === 'cover'` 渲染历史卡片列表
- **那么** 来源/元信息行容器**禁止**设置背景色（`background` / `bg-*` class）
- **且** 来源行的视觉独立性**必须**仅由顶部分割线 + 留白表达（原因：`CoverCard` 圆角容器与 footer 同级，背景色在圆角下方形成视觉割裂）

#### 场景:来源行与卡片主体水平对齐

- **当** `HistoryPage` 以 `cardStyle === 'cover'` 渲染历史卡片列表
- **那么** 来源/元信息行容器的左右内边距（`padding-left` / `padding-right`）**必须**与 `ComicCard` 主体（`CoverCard` 的 `p-2`）一致（均为 `px-2`）
- **且** 来源标签（`sourceSiteLabel`）与其后页码、章节、时间 span 之间的 `·` 分隔符间距**必须**保持现有 `mx-1` 不变

#### 场景:detailed 列表布局不受此需求影响

- **当** `HistoryPage` 以 `cardStyle === 'detailed'` 渲染历史卡片列表
- **那么** 来源标签作为同一 `text-xs` div 内的内联 `<span>` 渲染，用 `·` 分隔符串联页码/章节/时间
- **且** 行间分隔由父容器 `border-b border-[var(--border)]` 提供，**禁止**为此布局新增 footer 分割线或留白逻辑
