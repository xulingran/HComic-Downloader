# album-title-extraction 规范

## 目的
定义专辑标题提取的能力规范。提供纯函数 `extractAlbumTitle(titles)`（位于 `src/utils/titleSimilarity.ts`），从一组漫画标题中按分隔符 `/[\s\-—_～~]+/` 分词提取共有字段，作为"下载为专辑"弹窗的默认专辑名建议。函数必须为纯函数（无副作用、相同输入相同输出），禁止访问 React state 或发起 IPC 调用，且必须对原始标题（未经 `normalizeTitle` 去括号处理）操作以适配中文漫画标题的常见格式。
## 需求
### 需求:从多个漫画标题提取共有字段作为默认专辑名

系统**必须**提供一个纯函数 `extractAlbumTitle(titles: string[]): string | null`（位于 `src/utils/titleSimilarity.ts`），从一组漫画标题中提取共有字段，作为"下载为专辑"弹窗的默认专辑名建议。该函数**必须**为纯函数（无副作用、相同输入产生相同输出），**禁止**访问任何 React state 或发起 IPC 调用。函数**必须**对【原始标题】（未经 normalizeTitle 去括号处理）按分隔符 `/[\s\-—_～~]+/` 分词，分隔符集合**必须**覆盖空格、连字符（`-` `—`）、下划线（`_`）、波浪号（`～` `~`），以适配中文漫画标题的常见格式。

#### 场景:同一作品的不同章节提取出作品名主体
- **当** 传入 `["[作者A] 某作品 第1话", "[作者A] 某作品 第2话", "[作者A] 某作品 第3话"]`
- **那么** 返回 `"[作者A] 某作品"`

#### 场景:无作者前缀但作品名相同
- **当** 传入 `["某作品 第1话", "某作品 第2话"]`
- **那么** 返回 `"某作品"`

#### 场景:token 顺序不同但集合相同仍能提取
- **当** 传入 `["[作者A] 作品 第1话", "作品 第2话 [作者A]"]`
- **那么** 返回 `"[作者A] 作品"`（按集合交集判定，不依赖 token 顺序）

#### 场景:无空格的作者前缀仍保留作者
- **当** 传入 `["[作者A]某作品 第1话", "[作者A]某作品 第2话"]`
- **那么** 返回 `"[作者A]某作品"`（作者前缀单独判定，不依赖空格分词）

### 需求:分隔符扩展覆盖连字符等格式

分词分隔符**必须**包含连字符（`-` `—`）、下划线（`_`）、波浪号（`～` `~`），**禁止**仅按空格分词。这**必须**能正确处理共有部分出现在标题尾部、前缀各不相同的场景（如用连字符分隔的"系列名-子标题-作者"格式）。

#### 场景:连字符分隔且共有部分在尾部
- **当** 传入 `["偷袭观者-困困觉", "观者-结末-困困觉", "玄尘佛母-无相法身-困困觉"]`
- **那么** 返回 `"困困觉"`（三本前缀各不相同，但按 `-` 分词后交集为尾部共有的 `"困困觉"`）

#### 场景:空格与连字符混合分隔
- **当** 传入 `["作品名 第1话", "作品名-第2话"]`
- **那么** 返回 `"作品名"`

#### 场景:下划线作为分隔符
- **当** 传入 `["作者_作品_1", "作者_作品_2"]`
- **那么** 返回 `"作者 作品"`

### 需求:作者前缀强制保留

当且仅当【所有】输入标题都以同一个方括号片段（如 `[作者]`、`[社团]`）开头时，提取结果**必须**以该方括号片段作为强制前缀。该判定**必须**与"分词求集合交集"独立进行，**禁止**因作者片段不在分隔符切出的 token 内而丢失。若任一标题不以该方括号片段开头，则**禁止**添加作者前缀。当交集 body 已含作者片段（紧连如 `[作者]某作品`，或作为独立 token）时，**必须**去重，**禁止**输出 `[作者] [作者] ...` 这种重复形式。

#### 场景:全部标题共享同一方括号开头
- **当** 传入 `["[作者A] 作品X 第1话", "[作者A] 作品X 第2话"]`
- **那么** 返回值必须以 `"[作者A] "` 开头

#### 场景:仅部分标题带方括号开头
- **当** 传入 `["[作者A] 作品 第1话", "作品 第2话"]`
- **那么** 返回值**禁止**包含 `[作者A]` 前缀（交集为 `"作品"`）

#### 场景:作者前缀与连字符分词结合且作者紧连作品名
- **当** 传入 `["[社团X]系列A-第1话", "[社团X]系列A-第2话"]`
- **那么** 返回 `"[社团X]系列A"`（`[社团X]` 与 `系列A` 紧连无分隔符，作为单个 token；作者前缀已隐含，不重复加空格）

### 需求:提取失败或结果过短时返回 null

当输入标题数小于 2、集合交集为空且最长公共前缀长度小于 2、或提取结果 trim 后长度小于 2 时，函数**必须**返回 `null`，交由调用方决定回退文案。函数**禁止**返回空字符串或纯空白字符串。

#### 场景:仅一本漫画时返回 null
- **当** 传入 `["单本标题"]`
- **那么** 返回 `null`

#### 场景:完全无共有的标题返回 null
- **当** 传入 `["完全不同的标题甲", "毫无关联的标题乙"]`
- **那么** 返回 `null`

#### 场景:交集为空且公共前缀过短返回 null
- **当** 传入 `["第1话", "第2话"]`（公共前缀仅为 `"第"`，长度 1）
- **那么** 返回 `null`

#### 场景:提取结果过短返回 null
- **当** 提取出的共有部分 trim 后长度小于 2
- **那么** 返回 `null`

### 需求:集合交集兜底最长公共前缀

当按分隔符 `/[\s\-—_～~]+/` 分词求集合交集的结果为空时，函数**必须**回退到对【原始标题】求最长公共前缀（character-level），trim 后若长度 ≥ 2 则返回该前缀作为提取结果；否则返回 `null`。最长公共前缀**必须**在原始（未归一化）标题上计算，**禁止**使用去括号后的归一化结果（否则会丢失 `[作者]` 等有意义前缀）。

#### 场景:无空格但有公共前缀的标题
- **当** 传入 `["某作品第1话", "某作品第2话"]`（无空格，集合交集为空，但字符级公共前缀为 `"某作品第"`）
- **那么** 返回 `"某作品第"`

#### 场景:公共前缀为空白时返回 null
- **当** 传入 `[" 作品A", "\t作品B"]`（公共前缀为空白，trim 后长度 0）
- **那么** 返回 `null`
