# maintenance-scanner 规范

## 目的

定义维护中心扫描器对临时目录、来源推断与旧格式兼容的行为，确保维护功能在来源标识演进后仍能识别历史残留目录。

## 需求

### 需求:孤儿清理模式串更新

维护中心的孤儿临时目录清理模块中，用于推断 JM 来源的临时目录模式串必须使用 `temp_jm_*`，并继续兼容旧模式 `temp_jmcomic_*`。

#### 场景:孤儿目录来源推断

- **当** 维护中心扫描器扫描临时目录时
- **那么** 匹配 `temp_jm_*` 模式的目录必须被推断为 `"jm"` 来源
- **那么** 匹配 `temp_jmcomic_*` 模式的旧目录仍必须被视为 `"jm"` 来源
