使用说明

生成 PNG 图标

# 安装依赖（如果尚未安装） npm install sharp # 运行图标生成脚本 node scripts/generate-icons.mjs # 或者使用在线工具转换 SVG 到 PNG # https://cloudconvert.com/svg-to-png

在 Electron 中使用

// 图标已在 electron-builder.yml 中配置 // Windows: assets/icon.svg (自动转换为 ICO) // macOS: assets/icon.svg (自动转换为 ICNS) // Linux: assets/icon.svg (直接使用) // 运行构建命令 npm run build:win # Windows npm run build:mac # macOS npm run build:linux # Linux
